# 📄 Word to PDF Converter

A full-stack MERN application that allows users to upload Microsoft Word (.docx) files and convert them into PDF format through a simple and responsive web interface.

## 🚀 Live Demo

Link:
https://word-to-pdf-converter-topaz.vercel.app/

---

## ✨ Features

- 📤 Upload Word (.docx) files
- 📄 Convert Word documents to PDF
- ⬇️ Download the converted PDF
- 🎨 Clean and responsive UI
- ⚡ Fast file upload and processing

---

## 🛠️ Tech Stack

- React.js
- Express.js
- Node.js
- Axios
- Multer

---

## ⚙️ Installation

### Clone the Repository

bash
git clone https://github.com/Vishnu7011/Word-to-PDF-Converter.git


### Install Frontend

bash
cd frontend
npm install
npm run dev


### Install Backend

bash
cd backend
npm install
node index.js
